# FastP

Простая утилита для автоматической сортировки файлов в папке "Загрузки" (или любой другой) на Windows. Работает в фоне и раскладывает файлы по папкам в зависимости от их расширения.

![.NET Badge](https://img.shields.io/badge/.NET-8.0-512BD4?logo=dotnet&logoColor=white)

## Описание

FastP решает проблему беспорядка в папке загрузок. Программа висит в трее, следит за появлением новых файлов и сразу перемещает их в нужные категории (Картинки, Документы, Архивы, Аудио и т.д.).

Если файл еще скачивается браузером или занят другой программой, FastP подождет, пока он освободится, прежде чем перемещать.

## Возможности

*   **Автосортировка**: Мгновенно перемещает файлы при их появлении.
*   **Безопасность**: Не трогает файлы, которые используются или еще скачиваются.
*   **Настройки**: Можно менять правила сортировки и пути прямо в приложении.
*   **Фильтры**: Можно задать минимальный размер файла, чтобы не сортировать мелкий мусор.
*   **Трей**: Сворачивается в системный трей и не мешает работе.
*   **Статистика**: Показывает, сколько файлов было обработано за сегодня.

## Установка и запуск

Требуется Windows 10/11 и установленный .NET 8.0.

1.  Скачайте проект или клонируйте репозиторий:
    ```bash
    git clone https://github.com/Deithand/FastP.git
    ```
2.  Откройте папку проекта и запустите сборку:
    ```bash
    dotnet build
    ```
3.  Запустите приложение:
    ```bash
    dotnet run
    ```

## Как пользоваться

1.  Запустите FastP. По умолчанию он начнет следить за вашей папкой `Downloads`.
2.  Чтобы изменить настройки или добавить свои правила (например, кидать `.mp4` в папку `Видео`), нажмите кнопку "Настройки" или кликните правой кнопкой по иконке в трее.
3.  Если нужно временно отключить сортировку, нажмите кнопку "Пауза".

## Структура проекта

Проект написан на C# (WPF) с использованием паттерна MVVM.

*   `Services/FileSorterService.cs` — основная логика наблюдения за файлами.
*   `Services/ConfigManager.cs` — управление настройками и правилами (сохраняет в JSON).
*   `ViewModels/` — связка логики и интерфейса.
*   `Views/` — окна приложения (XAML).
